# 網路程式設計期中專案

|項目|內容|
|---|---|
|**專案名稱**|線上飲料點餐系統|
|**組員**|利蓁琳 陳冠華|
|**專案簡介**|使用者可瀏覽飲料菜單、查看詳細介紹、加入購物車、設定甜度冰塊、選擇取餐方式（自取／外送）、選擇門市、取餐時間並完成線上訂單送出。具備語系切換與深色模式。|
|**使用技術**|HTML、CSS、Bootstrap、JavaScript、DOM 操作、LocalStorage、Bootstrap Modal|
|**功能特色**|1. 飲料清單自動渲染與搜尋篩選<br>2. 飲料介紹 Modal 彈窗<br>3. 加入購物車／更新數量／刪除項目<br>4. 杯型、甜度、冰塊、價格即時計算<br>5. 自取／外送動態欄位顯示<br>6. 外送區域與門市自動對應<br>7. 取餐時間 Modal 選擇<br>8. 中英語系切換<br>9. 深色／亮色主題切換<br>10. 訂單驗證與錯誤提示|
|**加分項目**|localStorage、動畫效果、深色模式、RWD、互動功能|
|**GitHub Pages 網址**|https://github.com/Charlotte1107/114_web_midterm_project|
